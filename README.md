## ▶️ About

A **pixel-perfect YouTube UI clone** built from scratch using **Next.js 15** and **TypeScript**. This project replicates the full YouTube experience — homepage layout, video cards, sidebar, navbar, search bar, and responsive design — all without using the real YouTube API.

> 🎯 *The goal: clone one of the world's most complex UI systems — and make it indistinguishable from the real thing.*

---

## ✨ Features

| 🌟 Feature | 📖 Description |
|:---|:---|
| 🏠 **Homepage Layout** | Full YouTube homepage with video grid, sidebar & navbar |
| 🎬 **Video Cards** | Thumbnail, title, channel name, views & time — just like real YouTube |
| 🔍 **Search Bar** | Functional search UI with YouTube-style design |
| 📱 **Fully Responsive** | Perfect on mobile, tablet, and desktop |
| 🌙 **Dark UI** | YouTube's signature dark theme — pixel perfect |

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ or Bun

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Tcode-Motion/orchids-youtube-clone-project.git

# 2. Navigate into the project
cd orchids-youtube-clone-project

# 3. Install dependencies
npm install
# or
bun install
```

### Run Development Server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser 🎉

### Build for Production

```bash
npm run build
npm run start
```
